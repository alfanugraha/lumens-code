QuES-C Modul

Melakukan kuantifikasi perubahan cadangan carbon pada skala bentang lahan, yang bisa dikonversi menjadi besaran emisi dan besaran sekuestrasi dengan menggunakan metode Stock Difference

Parameter inpput

  1. Land use map-first time series and Land use map-second time series

    QUES-C melakukan analisa berdasarkan dua peta penutupan lahan seri waktu. Masukan datanya adalah peta tutupan lahan dalam format RASTER. Format data yang bisa digunakan adalah: GeoTiff (.tif), IMAGINE (.img), IDRISI Raster (.rst), GRID (.grd) dan ASCII raster (.asc).

  2. Planning unit map

    Peta unit perencanaan dalam format RASTER. Format data yang bisa digunakan adalah: GeoTiff (.tif), IMAGINE (.img), IDRISI Raster (.rst), GRID (.grd) dan ASCII raster (.asc).

  3. No Data value

    Kelas yang diberikan untuk No Data

  4. Carbon stock lookup table

    Tabel cadangan karbon untuk masing-masing tipe tutupan lahan. Format data yang bisa digunakan adalah CSV (.csv), TEXT file (.txt).

  5. Planning unit lookup table

    Tabel nama unit perencanaan dalam format RASTER. Format data yang bisa digunakan adalah CSV (.csv), TEXT file (.txt)

  6. Working directory

    Lokasi penyimpanan data hasil analisa QUES-C.

  7. Area Name

    Nama cakupan area yang akan dianalisa.

  8. Year of T1 and year of T2

    Waktu pengambilan data peta tutupan lahan seri waktu pertama dan kedua.

Hasil keluaran

  1. Carbon 1 & Carbon 2

    Keluaran berupa peta sebaran cadangan carbon pada area studi pada time series pertama dan kedua. Peta disimpan dalam format GeoTIFF(.tif)

  2. Emission

    Peta unit perencanaan dalam format RASTER. Format data yang bisa digunakan adalah: GeoTiff (.tif), IMAGINE (.img), IDRISI Raster (.rst), GRID (.grd) dan ASCII raster (.asc).

  3. Sequestration

    Tabel cadangan karbon untuk masing-masing tipe tutupan lahan. Format data yang bisa digunakan adalah CSV (.csv), TEXT file (.txt).

  4. Emission by zone

    Lokasi penyimpanan data hasil analisa QUES-C.

  5. Summary

    Nama cakupan area yang akan dianalisa.

  6. QuES-C Database

    Waktu pengambilan data peta tutupan lahan seri waktu pertama dan kedua.

  7. QuES-C Summary Report

    Waktu pengambilan data peta tutupan lahan seri waktu pertama dan kedua.
